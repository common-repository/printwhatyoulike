Places the PrintWhatYouLike button on blog posts and pages to generate a printer friendly version of the page.

== Description ==

The PrintWhatYouLike button generates a printer friendly version of your blog.  The reader will also have the option to save the page as a PDF, customize how the page is printed, or undo all the changes to page and restore the page to its original state.  This is great way for readers of your blog to save paper and ink when printing.

By default, the PrintWhatYouLike button will automatically format your blog for printing.  If you want to customize how your blog is formatted for printing, you can create a print layout for your blog.  Sign up for a *free* PrintWhatYouLike account at http://www.printwhatyoulike.com/users/new to create your print layout.

== Installation ==

1. Download the `printwhatyoulike.zip` file and unzip it.
1. Move the `printwhatyoulike` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set up your button through the 'Settings' -> 'PrintWhatYouLike' menu in WordPress
